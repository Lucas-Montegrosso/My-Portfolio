Proyecto 2

Este juego interactivo, desarrollado con JavaScript, desafía a los usuarios a adivinar un número secreto dentro de un rango establecido. El jugador recibe pistas si su suposición es demasiado alta o baja, brindando una experiencia entretenida y dinámica. La aplicación incorpora lógica de validación de entradas y retroalimentación visual para mejorar la interacción y hacer el juego más intuitivo.

Proyecto 3

Esta aplicación de validación de formularios, creada con JavaScript, garantiza que los datos ingresados por los usuarios sean correctos antes de ser enviados. Implementa verificaciones como la obligatoriedad de campos, formato de correo electrónico, longitud de texto y contraseñas seguras. Además, ofrece retroalimentación en tiempo real con mensajes de error personalizados, mejorando la experiencia del usuario.
